# Desafio Spotify (estudo)

Projeto simples que reproduz o visual de uma página estilo **Spotify**, criado apenas para fins de estudo.

## O que é
- Uma página estática que utiliza HTML, CSS e Bootstrap para layout e componentes.
- Não é uma aplicação conectada à API do Spotify — é apenas implementação visual e prática.

## Tecnologias
- HTML
- CSS (arquivo em `css/estilo.css`)
- Bootstrap (via CDN)

## Como usar
1. Abra o arquivo `index.html` no navegador (duplo-clique ou use um servidor local).
2. Os arquivos principais:
   - `index.html` — página principal
   - `css/estilo.css` — estilos personalizados
   - `img/` — imagens usadas

## Objetivo
- Praticar layout responsivo e componentes do Bootstrap.
- Aprender a organizar arquivos e estilos em um projeto front-end simples.
